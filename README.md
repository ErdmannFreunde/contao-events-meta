# Contao Events Meta

Fügt zu den Contao Events die Felder **Meta-Title** und **Meta-Description** hinzu und gibt diese auf der Event-Detail-Seite aus.
Werden die Felder nicht ausgefüllt, wird der Contao Standard verwendet.
